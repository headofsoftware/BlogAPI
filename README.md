<h3 style="color : red ">Projenin Amacı</h3>
<p>

<p>

</p>

<b>Proje canlıya çıkarıldığı zaman kullanıcıların yapmış oldukları işlemler kayıt altına alınacak. Bu sayede bir veriseti oluşturulacak.
